# 🏥 MASA Clinic Pro

> A Premium, Modern, and Complete Clinic Management System (CMS) designed for clinics, medical centers, and private practices. Experience state-of-the-art UI/UX with smooth transitions, customized controls, and powerful medical data modules.

---

## 🌟 Overview

**MASA Clinic Pro** is a professional desktop-based Healthcare Enterprise Resource Planning (ERP) application developed using C# and .NET Framework 4.7.2. With a sleek "Royal White & Accent Blue" theme powered by custom Guna.UI2 controls, it provides medical professionals with a premium workspace to manage patients, schedule appointments, handle billing, pharmacy inventory, and medical histories seamlessly.

---

MASA Clinic Pro is packed with powerful, production-ready modules:

## 💻 Tech Stack & Architecture

- **Front-End / UI**: 
  - Windows Forms (C#)
  - **Guna2 UI controls** (Sleek modern design, custom elipses, shadow panels, dynamic animations, borderless layouts).
  - High-Definition SVG/PNG dynamic assets.
- **Back-End / Engine**: 
  - .NET Framework 4.7.2
  - Linq-to-Entities
- **Database Engine**: 
  - **SQLite** database backend (`clinic.db`) for high-speed local data persistence.
  - Entity Framework 6 (EF6) for clean Object-Relational Mapping (ORM).
- **Reporting Engine**:
  - Custom Excel and PDF exportation modules.

## 🛠️ Getting Started & Installation

### Prerequisites
- Windows OS (Windows 10 / 11 recommended)
- **.NET Framework 4.7.2** runtime or developer pack
- Visual Studio 2019 / 2022 (with .NET desktop development workload)

### Compilation & Run
1. Clone or download the repository.
2. Open `Clinic Management System.sln` in Visual Studio.
3. Build the solution (Ctrl + Shift + B) to restore packages (Guna.UI2, EntityFramework, System.Data.SQLite).
4. Run/Start (F5) to launch the app!
5. On startup, the `clinic.db` database will automatically initialize, migrate, and seed initial test data.
